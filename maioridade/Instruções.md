**Lista de exercícios - Estruturas condicionais.pdf, pág. 14**

4 – Construa um algoritmo que solicita ao usuário que informe 4 idades. Depois, exiba quantas pessoas são maiores de idade (18 anos) e quantas são menores. Explique nos comentários.

**-- Passo a passo**
1 - Clonando o repositório inicial em qualquer local 

**git clone https://github.com/wcrvieira/algoritmos_js.git**

2 - Acessar a pasta do repositório baixado

**cd algoritmos_js\maioridade**

3 - Criar a estrutura do projeto Node Javascript

**npm init -y**

4 - Abrir a pasta do projeto no Visual Studio Code

**code -r maioridade**

5 - Criar o arquivo index.js no VSCode

6 - Escrever o algoritmo, verificar a sintaxe e a lógica.

7 - Compilar e testar o algoritmo no terminal do VSCode (CTRL + SHIFT + ')

**node index.js**

8 - Após a conclusão dos testes, subir a pasta do projeto no GitHub, seguindo os passos:
// Configurações globais do seu usuário GitHub

**git config --global user.name "seu_usuário"**

**git config --global user.email "seu_email"**


// Adicionando o seu repositório remoto

**git remote add origin <seu\_repositório\_github>**


// Adicionando os arquivos atualizados do projeto

**git add .**


// Comentários para confirmar os arquivos no Git

**git commit -m "mensagem inicial"**


// Enviando os arquivos do repositório local para o GitHub (remoto)

**git push -u origin main (ou master) --force**
